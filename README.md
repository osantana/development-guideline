# Development Guideline

Guidelines for Software Development Projects

## Setup

```
$ python -m venv .venv
$ .venv/bin/pip install -r requirements.txt
```

## Build

Sphinx can generate different formats. To see a list of options:

```
$ make
```

Then run:

```
$ make [format]
```
